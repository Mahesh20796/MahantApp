1. Project Overview

The Sabha Management System is a digital command center designed to manage community gatherings (Sabhas) with high efficiency. It streamlines member registration, automates attendance tracking, manages financial transactions (deposits/withdrawals), and provides insightful data analytics to leadership.

3. Module-Specific Analysis

3.1 Authentication & RBAC

The system implements Role-Based Access Control (RBAC). Depending on the user's role (Admin, Volunteer, Member), the application dynamically filters available features and restricts unauthorized actions. This ensures data integrity and security.

3.2 Dashboard (Analytics Hub)

The dashboard provides an immediate "at-a-glance" status of the organization. It features:

  • Real-time Counters: Total member count and current organization balance.
  • Trend Analysis: Weekly attendance line charts to track community growth.
  • Financial Tracking: Monthly balance trends to monitor fund sustainability.

3.3 Member Management

A centralized repository for all community members. Each member profile tracks personal details, contact information, role within the community, and their individual "wallet" balance for financial transparency.

3.4 Attendance Management

This module is integrated with the Sabha Schedule. It automatically suggests the most relevant Sabha session based on the current date. Marking attendance is a one-click process that saves records for all members simultaneously using efficient database upsert logic.

3.5 Wallet & Financials

Allows for individual transaction entries and Bulk Monthly Collections. When a transaction is saved, the system automatically recalculates the affected member's balance and updates the global organization fund.

4. Operational Procedures (How-To)

How to Manage Member Data

1. Go to Member Management.
2. Click "Add Member" to register a new person.
3. Use the Search bar to find existing members and click the "Edit" icon to update their profile or status.

How to Record Attendance

1. Navigate to Attendance.
2. Ensure the correct Sabha Date and Title are selected.
3. Select P (Present), A (Absent), or L (Leave) for each member.
4. Click Save Attendance at the bottom of the page.

How to Export Reports

1. Open the Reports module.
2. Select the desired date range.
3. Choose between PDF for professional presentation or Excel for data analysis.
4. Click the respective download button.

5. System Maintenance & Development

The system is built with a Service-Oriented Architecture. All database logic resides in supabase.service.ts. The application also features a "Mock Mode" which allows developers to test features even without an active internet connection by using local sample data.
